# Lunch Order Reserve (LOR) API

## คำอธิบาย
เป็นระบบที่จัดการหลังบ้าน ติดต่อกับ Database ใช้สำหรับการเชื่อมต่อกับระบบจองอาหาร (Front-End)

## ความต้องการระบบ
- Visual Studio 2019
- .Net Core 2.1
- Microsoft SQL Server 2014 or newer
